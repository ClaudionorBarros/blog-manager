About

Blog Manager is a web based software aimed for social/digital agencies to ease their contributors in posting to many blogs. Using PHP and Codeigniter as it's base, Blog Manager allows ease of extension for further development.

Features

For front-end users, posting a post is as easy as posting in the blog itself. No need to take a long time in populating many posts before creating a new one and simple media upload and embedding. For administration, Blog Manager has an .NET MVC like view. The current version is only support for Wordpress.
